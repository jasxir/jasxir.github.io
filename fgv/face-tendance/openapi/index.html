Save Attendance Record

Records a new attendance entry (Clock-in/Clock-out) for the worker.

path Parameters
workerId
required
string
Request Body schema: application/json
required
workerId
string

The unique worker ID (mapped from htad_worker_id).

date
string <date-time>

The timestamp of the attendance record (mapped from htad_date).

supervisorId
string

The ID of the supervisor who recorded the attendance (mapped from htad_supervisor_id).

attendanceCode
string

The status code for the attendance record, e.g., Present, Absent, Medical (mapped from htad_attendance_code).

Responses

Example FaceTendance Server

https://facetendance.fgvholdings.com/api/v1/workers/{workerId}/attendance
